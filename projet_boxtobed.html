PROJET : Modernisation du SI - BOXTOBED

Réalisé : Avril - Mai 2025
Projet BTS SIO - SISR

╔═══════════════════╗
║ CONTEXTE DU PROJE ║
╚═══════════════════╝
BOXTOBED est une chaîne d'hôtels à conteneurs souhaitant moderniser son SI.
Projet mené avec l'ESN INFOSUR sur une maquette virtuelle.

╔════════════════════════════╗
║ MISSION 1 – RÉSEAU ET DHCP ║
╚════════════════════════════╝
✔ Création de deux VLAN :
- VLAN 10 : utilisateurs – 192.168.10.0/24
- VLAN 100 : datacenter – 192.168.100.0/24
✔ Configuration du switch Cisco :
- Ports d'accès et trunk
✔ Déploiement du serveur DHCP sur SRV-DHCP
- Étendue 192.168.10.100–200
- Relay DHCP via ip helper-address sur le routeur

╔══════════════════════════════════════════╗
║ MISSION 2 – IMPORT AUTOMATIQUE DANS L’AD ║
╚══════════════════════════════════════════╝
✔ Script PowerShell développé (import_users.ps1)
- Lecture d’un CSV généré chaque jour à 6h
- Création des utilisateurs dans Active Directory si non existants
✔ Ajout d'une tâche planifiée à 6h10 chaque jour
✔ Gestion des doublons, mot de passe initial, OU par service

╔═════════════════════════════╗
║ MISSION 3 – SAUVEGARDE TFTP ║
╚═════════════════════════════╝
✔ Déploiement d’un serveur Debian 12 (SRV-TFTP)
✔ Installation de tftpd-hpa et configuration sécurisée
✔ Répertoire /srv/tftp en droits d’écriture pour les équipements
✔ Exemple de sauvegarde :
- Commande Cisco : copy running-config tftp:
- IP TFTP : 192.168.100.30 → fichier reçu et archivé

╔════════════════════════════╗
║ MISSION 4 – STRATÉGIES GPO ║
╚════════════════════════════╝
✔ Création de plusieurs GPO sur SRV-ADBOXTOBED :
- Mot de passe complexe (≥10 caractères)
- Fond d’écran d’entreprise (fond.jpg)
- BgInfo : infos système sur le bureau
- Interdiction de regedit
- Raccourcis par groupe : calculatrice (vente), bloc-notes (marketing)
- Redémarrage obligatoire après MAJ

╔═════════════════════╗
║ TESTS / VALIDATIONS ║
╚═════════════════════╝
✔ DHCP : les machines reçoivent bien une IP
✔ Script PowerShell : import correct sans doublons
✔ Sauvegarde switch : fichier TFTP confirmé
✔ GPO : toutes appliquées selon les groupes cibles

╔═══════════════╗
║ BILAN / RETEX ║
╚═══════════════╝
✔ Projet complet, toutes les missions validées
✔ Mise en pratique de compétences réseau, système et script
✔ Difficulté rencontrée : configuration du DHCP relay
Un projet formateur sur l’ensemble de la chaîne SI
