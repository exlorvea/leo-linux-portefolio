PROJET : Serveur Nextcloud sécurisé - r3st0.fr

Réalisé : 2024
Projet BTS SIO - SISR

╔════════════════════╗
║ CONTEXTE DU PROJET ║
╚════════════════════╝
Besoin de solution cloud privée pour le télétravail sécurisé et le partage de fichiers.
Choix : Nextcloud, open source et auto-hébergé.

╔═════════════════════════════════════════╗
║ MISSION – INSTALLATION ET CONFIGURATION ║
╚═════════════════════════════════════════╝
✔ Debian 11 sans GUI + Apache, PHP, MariaDB
✔ Installation de Nextcloud depuis archive ZIP
✔ Configuration d’un hôte virtuel Apache
✔ Sécurisation des droits et base SQL

╔═════════════════════╗
║ TESTS / VALIDATIONS ║
╚═════════════════════╝
✔ Accès web, upload, partage fonctionnels
✔ Compte admin actif et sécurisé
✔ SSH restreint au compte admin

╔══════════════════╗
║ BONNES PRATIQUES ║
╚══════════════════╝
✔ Utilisation minimale de services
✔ Droits stricts sur /var/www/nextcloud
✔ Prévision HTTPS (non encore déployé)

╔═══════════════╗
║ BILAN / RETEX ║
╚═══════════════╝
✔ Solution prête à l’usage réel
