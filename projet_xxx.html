PROJET : Active Directory

┌───────────────────────────┐
│ PROJET : Active Directory │
└───────────────────────────┘
Réalisé en : Mai 2024
Dans le cadre du BTS SIO – SISR (Atelier de professionnalisation)

╔════════════════════╗
║ CONTEXTE DU PROJET ║
╚════════════════════╝
L’entreprise fictive BOXTOBED souhaitait automatiser l’ajout de
postes clients dans son Active Directory. Le but était de simplifier
le déploiement en masse tout en respectant la hiérarchie OU/groupes.

╔═══════════╗
║ OBJECTIFS ║
╚═══════════╝
- Récupérer des données depuis un fichier CSV
- Générer un script PowerShell pour automatiser l’intégration
- Affecter les machines aux bonnes unités d’organisation

╔════════════════════════╗
║ COMPÉTENCES MOBILISÉES ║
╚════════════════════════╝
> Utilisation de PowerShell
> Connaissance de l’Active Directory (création d’objets, OU)
> Automatisation d’un processus IT
> Lecture de données CSV / gestion des erreurs

╔═════════════════╗
║ OUTILS UTILISÉS ║
╚═════════════════╝
> Windows Server 2022
> PowerShell ISE
> Bloc-notes ++ (préparation CSV)
> Machine virtuelle VirtualBox

╔════════════════════════╗
║ ILLUSTRATION / CAPTURE ║
╚════════════════════════╝

║ BILAN / RETEX ║
╚═══════════════╝
✔ Projet concluant : automatisation opérationnelle à 100 %
