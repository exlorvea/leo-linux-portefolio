PROJET : Infrastructure réseau - R3ST0.FR

Réalisé : 2023/2024
Projet BTS SIO - SISR

╔════════════════════╗
║ CONTEXTE DU PROJET ║
╚════════════════════╝
La société R3ST0.FR, spécialisée dans les avis de restaurants, a besoin de moderniser son infrastructure réseau afin de répondre à sa croissance rapide.

╔═════════════════════════════════════╗
║ MISSION 1 – ACTIVE DIRECTORY ET DNS ║
╚═════════════════════════════════════╝
✔ Installation de l'AD sur Windows Server 2019
✔ Création du domaine : r3st0.lan
✔ Configuration DNS associée
✔ Ajout de comptes utilisateurs et d’OU
✔ Intégration des machines au domaine

╔══════════════════════════╗
║ MISSION 2 – SERVEUR DHCP ║
╚══════════════════════════╝
✔ Installation du rôle DHCP
✔ Configuration de l’étendue : 192.168.1.100 à 192.168.1.200
✔ Configuration des options : DNS, passerelle, suffixe
✔ Attribution dynamique validée sur les clients

╔════════════════════════════════════════╗
║ MISSION 3 – SAUVEGARDE SWITCH VIA TFTP ║
╚════════════════════════════════════════╝
✔ Installation du serveur TFTP (tftpd-hpa) sur Debian 11
✔ Configuration sécurisée (/srv/tftp, chmod, systemctl)
✔ Sauvegarde via Netgear CLI → fichier transféré et restaurable

╔═════════════════════╗
║ TESTS / VALIDATIONS ║
╚═════════════════════╝
✔ Domaine : intégration réussie
✔ DHCP : attribution dynamique confirmée
✔ TFTP : fichier switch récupéré et restauré avec succès

╔═══════════════╗
║ BILAN / RETEX ║
╚═══════════════╝
✔ Projet complet, bonne gestion du réseau et des rôles serveur
✔ Difficulté : gestion des IPs doublons sur les VMs
Projet professionnalisant pour la gestion d’une petite infrastructure
